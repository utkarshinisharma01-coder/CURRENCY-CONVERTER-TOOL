# CURRENCY-CONVERTER-TOOL
# 💱 Currency Converter Tool  

A simple Currency Converter web application built using **HTML, CSS, and JavaScript**. It converts an amount from one currency to another using real-time exchange rates.

## Features

- Convert between different currencies
- Real-time exchange rates
- Simple and responsive design
- Easy to use interface

## Technologies Used

- HTML
- CSS
- JavaScript
- Exchange Rate API

```
│── index.html

```
